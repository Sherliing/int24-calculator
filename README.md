# int24-calculator

Калькулятор арифметических выражений на C с 24-битными числами со знаком,
представленными как массив из 3 байт.

## Возможности
- Поддержка операций +, -, *, /
- Поддержка скобок и приоритета операций
- Собственная реализация арифметики на уровне байт

## Алгоритмы
- **Умножение** — алгоритм Карацубы
- **Деление** — через последовательное вычитание
- **Вычисление выражения** — перевод в обратную польскую запись (через стек)
- Обработка отрицательных чисел через дополнение до двух

## О проекте
Учебная работа на изучение представления чисел на уровне байт и реализацию
базовой арифметики без использования встроенных типов int/long.
